## Quick start 🏄

Prerequisites: [Node (v21 LTS)](https://nodejs.org/en/download/) and [Git](https://git-scm.com/downloads)

> Clone the repository:

```
git clone https://github.com/salviega/certfieth-4-moodle-frame.git
```

> Install dependencies:

```
yarn
```

> Run the project:

```
yarn dev
```
